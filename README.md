# Example Use
For starters your should auth yourself with https://api.vps.net

    import vps.net
    vpsnet.init('your@email.me','your_auth_key')

After it you may do API calls:

    from vpsnet import cloudserver as cs
    sc.reboot('cloudserver_id')
